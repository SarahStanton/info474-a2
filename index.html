Assignment 2

Data from the World Happiness Report.

The different variables under examination are Happiness Score, Economy, Health, Freedom, and Trust. These variables are broken down by country, and year. I choose to encode these variables using area in the form of a bubble chart because it shows how much each variable changes for each country when the filters switches between years. Area also demonstrates how similar or dissimilar countries in the same region are by providing an easy comparison, although it is difficult to see small differences. Additionally, each country in a region was encoded with a different color to help differentiate between countries. A drawback of this method is that regions with fewer countries will have larger bubbles, which implies that they are happier overall than countries in other regions.

Year: This controlling element allows users to see change over time in specific variables for different countires. I learned that each variable hardly changes, if at all, from year to year in each country. It also allows you to compare which region has more change by comparing which region has more movement year to year.

Region: This controlling element allows users to see differences between specific countries within regions and compare overall patterns between regions. It is possible to see how variables vary, and compare this across region. All regions appear to have greater variability in Trust than in Happiness Score.

Variable: This controlling element allows users to see differences between different variables that influence happiness. I learned that some variables vary much more than others. Trust appeared to vary the most, and did not seem to be associated with either a higher or lower happiness score.
